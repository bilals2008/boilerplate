# Boilerplate: Next.js 16 + Express.js

A modern boilerplate for full-stack applications using **Next.js 16 (TypeScript)** for the frontend and **Express.js** for the backend. This setup provides a scalable architecture and developer-friendly environment for building production-ready applications quickly.

---

## Features

- **Frontend:** Next.js 16 with TypeScript
- **Backend:** Express.js
- **Full-stack setup:** Client and server separated for clean architecture
- **Error handling middleware** in Express
- **TantStack Query** integration (for data fetching)
- Ready-to-use **.gitignore** and project structure
- Modern **developer workflow** with clear folder separation
